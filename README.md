# express-microblog
A lightweight, server-side rendered microblogging application inspired by Twitter. This project was developed to demonstrate core backend mechanics, routing, and dynamic templating without relying on a frontend framework.

## Features
* **Server-Side Rendering (SSR):** Dynamically generates HTML using EJS templates.
* **Form Data Processing:** Captures and processes user input via HTTP POST requests.
* **Modular UI:** Utilizes EJS partials to maintain clean, reusable code.
* **Responsive Design:** Custom CSS using Flexbox to mirror the layout of modern social media feeds.
* **In-Memory State:** Stores session data dynamically before database integration.

## Tech Stack
* **Backend:** Node.js, Express.js
* **View Engine:** EJS (Embedded JavaScript)
* **Styling:** Custom CSS (Flexbox)
